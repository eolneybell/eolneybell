Bottleneck: A Browser-Based Coding Game
Bottleneck is a game that provides players with practice solving problems in Javascript. Indeed, designing this game also improved my skills in Javascript and CSS. Wireframes, heuristic evaluations, and usability tests were central to the development of this project.
bottleneck.world
Wireframes and mockups were built in AdobeXD
The game was built with a variety of tools, primarily React
Sprites, both static and animated, were designed for placement on the game's map
